《市场营销B》复习大纲(内容主要参考ppt、教材配合复习)

一：导论（见教材第一、二章内容）

（1）市场和市场营销定义

市场：具有特定需要或欲望，而且愿意并能够通过交换来满足这种需要和欲望的全部潜在和现实顾客。市场营销：市场营销是引导商品和服务从生产者到消费者或使用者的企业活动，以满足顾客并实现企业目标。

（2）市场营销的相关概念（需要、交换、需求、价值、顾客满意等）、顾客让渡价值

需要：是指个人没有得到某种满足的感受状态。需求：是指人们对某个产品有购买欲望且有支付能力。

交换：是指从他人处取得所需之物，而以某些东西作为回报，是一个价值创造过程，通常总使双方变得比交换前更好。

价值：顾客感知价值:相对于竞争者，顾客对一种营销供给物总利益与总成本之差所作出的评估。顾客通常并不是精确或客观地产品价值和成本，而是依据感知价值判断。

顾客满意：顾客对产品的感知价值与他的期望值比较后所形成的愉快或失望的感觉状态。顾客让渡价值：顾客总价值与顾客总成本之间的差额。

（3）市场营销观念的发展变化

被顾客驱动的营销——-顾客驱动的营销

二：市场营销环境分析（见教材第四章、微观环境中的竞争者见教材第九章内容P149）

（1）市场营销环境分析概述

市场营销环境（含义）：存在于企业营销系统外部的不可控制或难以控制的各种影响企业营销活动及其目标实现的外部条件、因素和力量。

营销环境的特征：客观性、多变性、差异性、相关性

（2）市场营销宏观环境和微观环境分析因素

市场营销微观环境：对企业服务其顾客的能力构成直接影响的各种力量，包括企业本身、供应商、营销中介、市场、竞争者和各种公众，这些都会影响企业为其目标市场服务的能力。

市场营销宏观环境：指对企业营销活动造成市场机会和环境威胁的主要社会力量。1.人口环境2.经济环境3.自然环境4.科学技术环境5.政治法律环境6.社会文化环境

（3）相关的环境分析工具

五力模型：现有行业竞争者的威胁+潜在进入者威胁+替代品的威胁+供应商的谈判力+购买者的谈判力

机会威胁分析：

  • 环境威胁：环境中不利于企业营销的因素及其发展趋势。

  • •市场机会：环境变化形成的对企业营销活动富有吸引力和利益空间的领域。

SWOT分析：优势（Strength）、劣势（Weakness）、机会（Opportunity）和威胁（Threat）

SW:企业自身的资源条件、实力分析。

OT:企业外部环境的变化及对企业的可能影响。

市场营销环境的特点：客观性、差异性、相关性、替代性

三：消费者市场及其购买行为（见教材第六、七章内容）

（1）理解消费者市场和消费品，了解消费者市场和组织市场的差异

消费者市场：所有为了满足个人消费而购买产品和服务的个人和家庭所构成的市场。

组织市场：以某种组织为购买单位的购买者所构成的市场，购买目的是为了生产、销售、维持组织运作或履行组织职能。

组织市场的构成：组织市场是指所有为满足其各种需求而购买产品和服务的组织机构所构成的市场。

消费者市场特点：

  1. 分散性
  2. 差异性
  3. 多变性
  4. 替代性
  5. 非专业性

组织市场特点：

  1. 派生需求
  2. 过程复杂
  3. 多人决策
  4. 提供服务

消费者市场和组织市场的差异：

  1. 主要消费对象不同：消费品市场是指个人或家庭为满足生活需要而购买商品或服务的市场。组织市场是指企业事业单位政府社团为生产办公或者服务需要购买商品或服务的市场。
  2. 市场需求不同：消费品市场需求是直接性的，是由消费者的直接需求产生的。组织市场需求是一种派生类的需求，是由消费市场派生出来的，因而最终也取决于消费市场需求。
  3. 需求弹性不同：虽然消费品市场和组织市场的需求弹性都较大，但是消费品市场需求弹性会因为价格、品种、规格、样式、品质、功能等因素上下浮动，已发生变化。组织市场的需求波动则是因为市场需求波动变化而变化，并且这种变化在短期内不可能回复或者反向波动。
  4. 规模、结构以及地理分布不同：消费品市场购买人数多，每次购买的商品数量少，因而市场交易次数频繁，交易场所多样。组织市场买主户数少，规模大的特点和自然条件、交通设施、市场竞争、历史原因等因素的影响导致产业分布趋向集中，组织市场买卖双方关系稳定，并常开展互购。

（2）刺激-反应模型，影响消费者行为的因素

注意—了解—接受

影响消费者行为的因素：

（3）消费者决策模式的分类

1.复杂型购买行为：消费者专门仔细的购买,并注意现有各品牌之间的差异时,就是在从事复杂的购买行为。（贵重物品、大型耐用品、风险较大的商品）

2.变换型购买行为：消费者了解现有各品牌和品种之间的明显差异，但并不深入收集信息,购买时随意性较大，只在消费后才加以评估，但是在下次购买时可能又会转换其他品牌。

3.协调型购买行为：消费者购买差异性不大而购买风险较大的商品时所发生的一种购买行为。（服装、首饰、家具、小家电等）

4.习惯型购买行为：卷入程度不高而且品牌差异也不大，消费者不是按照决策过程实施购买行为，而是不假思索直接购买习惯的产品。（价格低廉、易耗品）

四：目标营销战略（见教材第八章内容）

（1）市场细分的内涵，依据与原则

市场细分的内涵：也被称作市场细化，是指在市场调查的基础上，根据消费者的需求、购买习惯和购买行为的差异性，以及企业自身资源限制把整个市场细分为若干个市场部分或者亚市场的过程。

每一个细分市场都是一个有着相似欲望和需要的消费群，而分属不同细分市场的消费者的欲望和需要存在明显差异。

市场细分的依据：地理变量、人口变量、心理变量和行为变量，以及其组合效用细分等。

市场细分的原则：

  1. 可区分性：不同的子市场在概念上可以明确区分，如：女性化妆品市场可以依据年龄层次和肌肤类型等变量加以区分。
  2. 可测量性：细分市场的子市场的大小及其购买力的数量资料能够加以测量和推算，如：电冰箱是市场，有多少人更注重价格、有多少人更注重能耗、外观或兼顾等。
  3. 可盈利性：细分后的市场有足够的需求潜量，且有一定的发展潜力，其规模足以让企业有利可图。
  4. 可进入性：企业细分后的子市场应能够借助营销努力达到进入的目标，组合营销策略能够在该市场上发挥作用。

（2）目标市场选择，目标市场策略

目标市场的五种选择模式：

  1. 产品/市场集中化
  2. 产品专业化
  3. 市场专业化
  4. 多个细分市场专业化
  5. 市场全面化

选择目标市场营销战略考虑的因素：

  1. 企业资源
  2. 产品同质性
  3. 市场同质性
  4. 产品所处的生命周期
  5. 竞争对手的目标市场涵盖战略

目标市场涵盖战略：

  1. 无差异性营销战略
  2. 差异性营销战略
  3. 集中性营销战略
  4. 微市场策略（微市场营销，指企业根据特定个人或特定区域的偏好调整自己的产品和营销策略，包括当地营销和个人营销1区域化或定制化经营，能满足特定顾客的需求。2更高的利润空间。3更紧密的互动。4经营者承担风险较大。）

（3）市场定位的含义、步骤、策略

市场定位的含义：指企业针对潜在顾客的心理进行营销设计、创立产品、品牌或企业在目标顾客心目中的某种形象或个性特征，保留深刻的印象和独特的位置，从而取得竞争优势。竞争优势——鲜明特色、深刻印象

市场定位的步骤：

  1. 确认本企业的竞争优势
  2. 准确地选择相对竞争优势
  3. 明确显示独特的竞争优势

市场定位的依据：

  1. 产品属性定位
  2. 竞争者定位：以竞争者品牌为参照物并依附其定位。目的是通过品牌竞争提升自身品牌的价值与知名度。
  3. 使用场合定位
  4. 顾客利益定位
  5. 使用者定位

市场定位的方法：

  1. 初次定位
  2. 重新定位
  3. 对峙定位
  4. 避强定位

五：4P-产品策略（见教材第十、十一章内容）

（1）产品整体概念

所谓产品，是指能提供给市场，用于满足人们某种欲望和需要的任何事物，包括实物、服务、场所、组织、思想、主意等。

营销学中关于产品的陈述有一句非常著名的话：“你卖的不是一个钻头而是一个洞”。意思是说企业生产的产品或提供的服务只是满足消费者的某一特定需求的工具或手段，消费者购买的并不是你的产品本身，而是他的某一需求或利益满足。

一个完整的产品概念是立体的，包括核心产品、形式产品、附加产品三个层次。

（2）产品组合以及产品组合的优化

产品组合(ProductMix)，企业生产经营的全部产品的结构或者比例，即某一企业所生产或销售的全部产品线和产品项目的组合。

优化产品组合的过程通常是分析、评价和调整现行产品组合的过程，包括：

  1. 产品线销售额和利润分析——销售增长率、利润率
  2. 产品项目市场地位分析——市场占有率/相对市场份额

（3）产品生命周期理论

产品市场生命周期的概念：产品从进入市场到退出市场所经历的时间阶段。包括：导入期、成长期、成熟期、衰退期。

生命周期的阶段划分：1.类比法2.销售增长率法3.普及率法

（4）品牌策略：品牌和商标的联系和区别，掌握品牌策略。

品牌的概念：品牌是一种名称、术语、标记、符号或设计，或是它们的组合运用，其目的是借以辨认某个销售者，或某群销售者的产品及服务，并使之与竞争对手的产品和服务区别开来。商标，品牌或品牌的一部分注册后，受法律保护。

_商标掌握在企业手中，而品牌是属于消费者的！！_

品牌基本策略：

  1. 品牌有无策略
  2. 品牌归属策略
  3. 品牌统分策略（个别品牌、统一品牌、分类品牌、企业名称+个别品牌）
  4. 品牌发展策略

七：4P-分销渠道策略（见教材第十三章内容）

（1）分销渠道概述、渠道结构的两个维度

分销渠道：分销是指帮助企业实现产品或服务，以及有关所有权从生产者到消费者的转移；

分销渠道是指产品从生产者向消费者转移过程中，取得产品的所有权或帮助所有权转移的所有企业和个人；——包括生产者、经销商、代理商和最终消费者；

营销渠道是指配合或参与生产、分销和消费某产品的所有企业和个人；——包括供应商、生产者、经销商、代理商、辅助商和最终消费者。

营销渠道成员=供应商+分销渠道成员+辅助商

渠道结构：

  • 按有无中间商分：直接渠道和间接渠道；
    • 复合渠道：直接渠道和间接渠道相结合；
  • 按中间商层次数量分：长渠道和短渠道；

按渠道每个层次选择同种类型中间商的数量分：宽渠道和窄渠道；

考虑产品性质、市场特征和企业分销战略等因素,宽渠道还可以划分为：独家分销渠道；选择分销渠道；密集分销渠道

（2）分销渠道设计决策，影响分销渠道选择的因素

分销渠道的设计：

  1. 确定渠道目标与限制
  2. 明确各种渠道方案
  3. 评估各种可能的渠道方案

影响分销渠道选择的因素：

  1. 顾客
  2. 产品
  3. 中间商
  4. 竞争者
  5. 企业
  6. 环境

（3）渠道冲突

渠道冲突是指渠道系统中某些成员因从事妨碍企业实现目标的活动而发生各种矛盾和纠纷的现象。

渠道冲突的产生原因：目标差异；归属差异；认知差异；过度依赖

渠道冲突的类型：水平渠道冲突；垂直渠道冲突；多渠道冲突

渠道冲突管理的主要内容：

（1）预防和避免冲突；

（2）控制冲突水平，避免不良冲突发生；

（3）利用冲突资源，激励渠道成员；

（4）化解冲突危机，舒缓渠道合作关系；

（5）切断冲突源头，调整渠道关系。

八：4P-促销策略（见教材第十四章内容）

（1）促销策略概述——了解促销组合要素的构成

产品类型、推式与拉式策略、促销目标、产品生命周期阶段、经济前景

（2）广告

概念：广告是由明确的发起者通过各种付费媒体所进行的各种非人员的或单方面的沟通形式，是对产品、服务或某项行为的意见和想法等的介绍。

广告策略的主要内容：

程序化广告：鉴于广告管理成本和时间成本的高昂，程序化广告逐渐成为许多热门选择。

程序化广告是指利用技术手段进行广告交易和管理。广告主可以程序化采购媒体资源，并利用算法和技术自动实现精准的目标受众定向，只把广告投放给对的人。媒体可以程序化售卖跨媒体、跨终端(电脑、手机、平板、互联网电视等)的媒体资源，并利用技术实现广告流量的分级，进行差异化定价。因此，程序化广告是一种使用AI根据定位算法购买、投放广告的过程。2020年，超过86％的数字广告通过自动渠道购买。

程序化广告为广告主和媒体提供高效、智能、精准的营销机会，带来营销效率和广告效果的双提升。此外，程序化广告超越了网络空间，还可以用于购买和销售各种媒体广告资源，其中包括展示广告到户外广告。

程序化和传统广告之间的关键区别在于程序化可以实时更新，使得广告商掌握更多可控性。

即使是中小卖家也可以利用程序化广告来最大化投资回报率。

而使用程序化广告的方式则决定了程序化广告的成功与否。

程序化广告依赖于自动化、大数据、媒体技术和专业知识的结合，而这无疑需要投入一定的人力成本

（3）人员推销

概念：推销是企业派出销售人员，直接向可能的购买者进行的推销活动。是向消费者宣传、介绍产品，说服消费者购买，向消费者提供其真正需要的满意产品的一种销售行为。

（4）销售促进（营业推广）

概念：所谓销售促进，是指企业运用各种短期诱因，鼓励消费者购买或经销商销售企业产品或服务的促销活动。

分类：

针对消费者的促销工具——样品、优惠券、以旧换新、打折降价、赠奖、商品示范等

针对产业用品的促销工具——折扣、赠品、特殊服务等

针对中间商的促销工具——购买折让、免费货品、商品推广津贴、合作广告、推销金、经销商销售竞赛等

针对推销人员的促销工具——红利、竞赛、培训指导等

随着网上订单的便捷化和低成本化，团购成了许多商家促进销售的利器。

团购是团体购物的简称。它是指认识或不认识的消费者联合起来，提高与商家的谈判能力，以求得最优价格的一种购物方式。

商家以低于零售价格的团购折扣和单独购买得不到的优质服务来吸引消费者，通过消费者自行组团、专业团购网站、商家组织团购等形式，来促进订单增长并实现产品和服务的推广。

（5）公共关系

概念：公共关系（publicrelations）是指某一组织为改善与社会公众的关系，促进公众对组织的认识、理解及支持，达到树立良好的组织形象、实现组织与公众的共同利益与目标的管理活动与职能。

对象：对外、对内的公关关系的对象

活动方式：赞助公益活动、新闻演讲、利用大型国际活动、听取和处理公众意见、建立与各种公共关系对象的良好关系

公关目标：通过获得公众赞同来与公司的利益相关者建立良好的关系，树立良好的公司形象，处理或消除不利的传言、事件等。
